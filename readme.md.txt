# RentX 🏡

A full-stack property rental platform built with **React** and **Go**, designed for speed, simplicity, and scalability. With RentX, users can browse listings, and property owners can easily post rental offers.

---

## 🔧 Tech Stack

**Frontend**
- React
- React Router
- Tailwind CSS
- Vite

**Backend**
- Go (Golang)
- SQLite (via `modernc.org/sqlite`)
- REST API
- File upload support
